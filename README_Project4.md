## 🎯 Dashboard Design Principles Applied

| Principle | How Applied |
|-----------|-------------|
| Form follows function | Chart types match business questions |
| No 3D effects | All charts are flat 2D |
| Axis integrity | All bar chart axes start at zero |
| Max 3 pie slices | Order status pie uses 3 grouped categories |
| Direct labeling | Data labels on all charts |
| Data-ink ratio | Chartjunk removed, clean backgrounds |
| Action titles | Every chart title states the conclusion |
| SCR Framework | Insights → Situation, Complication, Resolution |
| 5-second rule | KPIs placed top-right for instant reading |
| "So What?" answered | Recommendations box addresses next steps |

## 💡 Key Insights

- ⚠️ **41.4% of orders are cancelled or returned** — This is the single most critical finding. Cancelled (20.8%) + Returned (20.6%) = urgent business investigation needed
- 📱 **Instagram drives highest revenue** — ₦275,285 and 259 orders — top marketing channel by both volume and revenue
- 💰 **Chairs & Printers generate equal highest revenue** — ₦195K each despite different order volumes
- 🏦 **Payment methods are balanced** — No single method dominates, indicating healthy customer payment diversity

---

## 🎯 Recommendations

1. **Investigate the 41.4% at-risk rate** — Conduct root cause analysis on cancellations and returns to identify if it's product quality, delivery, or expectation issues
2. **Increase Instagram marketing budget** — Highest ROI channel; additional investment should yield proportional returns
3. **Upsell Laptops to existing customers** — Laptops have higher average order value despite lower volume — target existing customers with upgrade offers
4. **Expand referral program** — Referral channel is currently the weakest; a structured referral incentive program could improve performance

## 🧠 Concepts Learned
- Data visualization is a **moral act** — misleading charts destroy credibility
- Every chart must answer **"So What?"** to be useful to decision-makers
- **Whitespace is not wasted space** — it frames data and reduces cognitive load
- The **5-second rule** — an executive must understand the key insight within 5 seconds
- **Color is a tool for highlighting**, not decoration — use sparingly and purposefully
